Welcome to r-balls!

An r-ball contains linked data and Semantic Web representations of cultural heritage resources described using RDA: resource description and access.

An r-ball is created and maintained using RIMMF (RDA in Many Metadata Formats). Try RIMMF-balls for newbies.

R is for RIMMF/RDA/RDF/Resource/Record!

* R-balls are the primary data for a Jane-athon.
